📥 WordPress-এ Import করার নিয়ম

  1. WordPress এ যেকোনো Page এর Elementor Edit খুলুন
  2. নিচে বাম কোণে Folder Icon (Import Template) এ ক্লিক করুন
  3. My Templates → Import Template ক্লিক করুন
  4. Download করা .json file টি upload করুন
  5. Insert চাপুন — Done! ✅
